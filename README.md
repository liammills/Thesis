## How to run

**Note**: If on an M1/M2 chip, install Rosetta 2 with `softwareupdate --install-rosetta` to ensure compatibility with AMPL modules.

### Using Poetry
1. `poetry shell`
2. `poetry install`
3. `cd hems`
4. `python3 hems_pv_battery`
5. `exit` to exit poetry shell
